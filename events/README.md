# イベント

このワークショップの目的はGoとGoogle App Engineを使用して、
ゼロからwebアプリケーションを作成してデプロイすることです。

このアプリケーションは各イベントの天気情報を持つイベントマネージャーです。 クール、そうですか？
それは驚くべきことではありませんが、しかしですね ... 私は*本当に*フロントエンドエンジニアではありません。 😅
